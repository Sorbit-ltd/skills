# Sorbit Skills

Agent Skills for the Sorbit platform — search courses and upload structured content to Sorbit's AI tutoring system.

## Install

Run the following in Claude Code:
```
/plugin marketplace add Sorbit-ltd/skills
/plugin install sorbit-courses@Sorbit-ltd/skills
```

## Skills

### `sorbit-courses`

Search Sorbit's public course catalogue and upload structured courses to the platform via the Sorbit API.

- **Search** — no authentication required
- **Upload** — requires an API key from the [Sorbit Creator Hub](https://sorbit.co.uk)
